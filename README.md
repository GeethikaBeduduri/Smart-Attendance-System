# Smart Attendance System

## Overview
An AI-based Smart Attendance System that uses Face Recognition to automatically identify users and mark attendance.

## Features
- Face Detection using OpenCV
- Face Recognition
- Automatic Attendance Marking
- Attendance Record Management
- User-Friendly Interface

## Technologies Used
- Python
- OpenCV
- NumPy
- Face Recognition
- Haar Cascade Classifier

## Project Structure
- main.py
- haarcascade_frontalface_default.xml
- install commands.txt

## How to Run

1. Install required libraries
2. Run:
   python main.py
